# Tutorials

Learning-oriented lessons. Each one is a guided, hands-on path with a concrete
result at the end. Work through them in order if you are new.

- **[Getting started](getting-started.md)** — build the mod, drop it into a Fabric
  server alongside CC:Tweaked, and run your first `wasm` Lua snippet.
- **[SQLite walkthrough](sqlite-walkthrough.md)** — drive a real `sqlite3.wasm`
  module through the raw WebAssembly API (mode A): open an in-memory database,
  create a table, insert rows, and read results back.

Once you are comfortable, move on to the task-focused
[How-to guides](../how-to/index.md) or the complete
[Reference](../reference/index.md).
